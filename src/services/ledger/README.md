# Ledger Service

Immutable transaction ledger and audit trail for MAYA Node.

## Purpose

Provides tamper-proof record-keeping:
- Transaction logging
- Audit trail
- Integrity verification
- Chain validation

## Key Components

- `ledger.py` - Core ledger implementation
- `transaction.py` - Transaction definitions
- `validator.py` - Chain validation
