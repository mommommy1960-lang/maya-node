# MAYA Node – Datasheet

**Use Cases:**  
Remote cell sites, emergency shelters, clinics, fleet depots, edge compute.

**Inputs:**  
- PV (up to 1,000 Vdc strings)
- Small wind (rectified)
- Optional genset, grid

**Storage:**  
- LFP battery modules, UL9540A-tested  
- Optional supercapacitor rack

**Outputs:**  
- 120/240 Vac split-phase or 208/480 Vac 3-phase via transformer
- 48 Vdc bus

**Brains:**  
- Edge controller: predictive dispatch, fast load shed, SOC-aware curtailment

**Thermal:**  
- Liquid loop to 100–300 L PCM or hydronic coil

**Safety & Compliance:**  
- UL 1741 SB, IEEE 1547, UL 9540  
- Type 3R/4X enclosure

**Telemetry:**  
- Local HMI, Modbus TCP, OPC UA, opt. cellular

**Footprint:**  
- 2 outdoor cabinets + combiner  
- Forklift pockets and pad mounts

**Service:**  
- 30-min. swap modules, remote FW updates, 5-year O&M

**Key Metrics:**  
- 60–90% fuel offset vs. diesel-only  
- <50 ms transfer priority loads  
- 88–94% AC↔AC round-trip eff.  
- Hot-swappable 10–50 kW inverters, 50–200 kWh packs
